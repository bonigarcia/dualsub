DualSub ha sido desarrollado en Java y dispone versiones para:


Windows

Instalador para Microsoft Windows. Asistente que instalará DualSub en Archivos de programa.


Descargar

Mac OS X

Instalador para Mac OS X. Abre el archivo DMG y arrastra DualSub a Aplicaciones.


Descargar

GNU/Linux

Paquete debian. Usa el siguiente comando para instalar DualSub: 

dpkg -i dualsub-1.1.0.deb

Descargar

Cross-platform

Archivo JAR ejecutable. Ejecuta DualSub usando el siguiente comando: 

java -jar dualsub-1.1.0.jar
